IDEA使用技巧与经验

IDEA基础技巧

IDEA基本知识

  • 新安装的IDEA是没有工具栏的，可以通过 view 打开工具栏 toolbar
  • IDEA是自动保存的，不需要 Ctrl + S
  • Alt + Enter 是快速修复代码
  • Alt + Insert 是快速引入(生成代码、Maven搜索Jar坐标)
  • Ctrl + Enter 是SQL执行当行语句
  • IDEA并非自动编译的，可以通过设置修改为自动编译，或手动编译 Ctrl + F9
  • IDEA的视角为 工程 – 模组，且配置有 项目配置 与 IDEA配置

IDEA常用配置：

  • 设置编码格式(默认跟随系统GBK)：
    Settings –> Edittor –> File Encodings–//–各种设置
  • 设置鼠标滑轮调整代码字体大小：
     Settings –> Edittor –//– Mouse –> Change font size….
  • 设置鼠标停留显示提示：
     Settings –> Edittor –//– Other–> Show quiqck documentation…
  • 设置自动导包：
     Settings –> Edittor –> General –> Auto Import –//– Java–> Add unambiguous… 和 Optimize import…
  • 设置显示代码行号：
     Settings –> Edittor –> General –> Appearance –//– Show line numbers
  • 设置方法分隔符：
     Settings –> Edittor –> General –> Appearance –//– Show method separators
  • 关于代码提示的大小写区分：
     Settings –> Edittor –> General –> Code Completion –//– Match case
  • 设置文件导航多行显示tabs：
     Settings –> Edittor –> General –> Editor Tabs –//– Appearance –> show tabs in one row
  • 设置Maven随时更新pom坐标：
     Settings–>Maven–>Importing –//– Import Maven projects automatically
  • 设置编写模板：
     Settings –> Edittor –> Live Templates–> 设置
  • 设置后缀生成：
     Settings –> Edittor –> General –> Postfix Completion

各种字体配置

  • 字体设置(默认)：
     Settings –> Edittor –> Font –//– 设置
  • 字体设置(编辑区)：
     Settings –> Color Scheme –> Color Scheme Font –//– 设置
  • 字体设置(控制台)：
     Settings –> Color Scheme –> Console Font –//– 设置
  • 字体设置(注释)：
     Settings –> Color Scheme –> Language Defaults –//– Comments –> 各种Comment设置

项目配置

  • SDK设置（软件开发工具）：
     Project Structure –> Plaform Settings –> SDKs –//– 设置

IDEA快速编码

快捷键

由于自己是Eclipse转过来IDEA的，所以基本快捷键映射修改为了Eclipse

代码自动补全： code-completion-basic 更改为 Alt + / 但会有重复占用，删除原来的占用
自动生成代码： Alt + Insert
处理错误： Alt + Enter
重命名： Alt + Shift + R
自动补全本行： Ctrl + Shift + Enter
打开模板提示： Ctrl + Alt + Shift + J

格式化代码： Ctrl + Alt + L 、Ctrl + Alt + F
代码缩进： Ctrl + Alt + I
方法参数提示： Ctrl + P
嵌套选中代码： Ctrl + Alt + T

查找锁定位置： Ctrl + J
方法之间快速切换： Ctrl + Shift + Up/Down

查找类： Ctrl + Alt + Shift + N
替换文本： Ctrl + F
显示最近操作的文件： Ctrl + E

常用模板

输出： .sout
输出错误： .serr
输出某参数： soutv
输出传入参数： soutp
输出类名+方法： soutm

Main方法： psvm
静态变量： psfi
判断是否为null： .nn .null
判断该表达式： .if .else
声明变量： .var 类似快捷键Ctrl + Shift + L
循环： .fori
遍历集合ForEach： 集合对象.for

文件夹标注

  • 通过文件夹右键或者项目配置，设置文件夹类型：source resource java 等等，不同标记其权限及使用不同

IDEA使用过程中遇到的问题

在任务栏会一直闪烁，不停提示

  1. 选择File，选择Invalidate Caches/Restart，选择Invalidate and Restart
  2. IDEA会重启，重启过后正常
