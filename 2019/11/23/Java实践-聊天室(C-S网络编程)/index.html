Java实践-聊天室(C/S网络编程)

项目解析

项目练习

  • 熟悉Java网络编程
  • 练习通过多线程完成多任务

项目需求

  • 编写聊天室，客户端与服务端
  • 可是实现多客户端同时连接，同时群聊

需求分析

  • 通过Java网络编程实现网络连接：使用ServerSocket和Socket两个类
  • 通过Java多线程技术实现多客户端连接：使用Thread类
  • 多连接，多数据传输，运用流技术实现实时传输

项目搭建-客户端

客户端实现的功能

  • 不断等待发送数据
  • 不断等待接收数据

客户端代码实现

项目搭建-服务端

服务端实现的功能

  • 连接成功后向服务端发送欢迎消息
  • 不断等待接收数据，接收成功后，向正在连接的每个客户端转发消息
  • 要有统计、管理客户端的能力

服务端代码实现

项目总结

难点

  • 对于Java网络编程的熟悉要求
  • 客户端相对的简单，只需要发送与接收
  • 服务端相对复杂，接收与群发，还有列表管理的支持

技巧

  • 巧用线程，实现多任务
  • 巧用ArrayList，方便管理列表，泛型可以选择Socket也可是PrintWriter，项目中使用PrintWriter方便群发
