Eclipse基础技巧

Eclipse配置：

字体大小：

显示行号：
Preferences(偏好)--Editors(编辑器)--Text Editor--//--Show line numbers -- checked
取消News：
Preferences(偏好)--News(新闻)--//--Enable automatic news polling -- Unchecked

Eclipse代码提示：

Preferences(偏好)--XML--XML Files--Editor(编辑器)--Content Assist(代码补齐)--(<=:abcdefghijklmnopqrstuvwxyz)

Eclipse插件使用技巧

jad反编译插件

  1. Help –> EclipseMarketplace –> Find:jad
  2. 一路确定重启Eclipse就可以了
  3. 这是在线安装方式，也可离线安装，但没有成功
  4. 可以反编译一些class来探究编译器的编译过程

Eclipse某些错误更改

Source Not Found问题

  1. 从windows选项下选择”perferences”
  2. 在”perferences”对话框下选择java->Installed JREs
  3. 选中”jreX”并点击右侧”Edit”
  4. 在”Edit JRE”对话框中找到对应的jar，例如rt.jar，点击展开选择”Source Attachment”
  5. 在”Source Attachment”对话框下，选择”External location”，路径设为JDK根目录，源文件就是该路径下的src.zip
