MarkDown通用语法

一、标题

一般使用 #空格 来进行层级标识。共 6 个层级，再多不识别。
　　# => h1;
　　## => h2;
　　### => h3;
　　#### => h4;
　　##### => h5;
　　###### => h6;

注：h1 级别会默认带一个 <hr/>

二、分隔线

至少三个 * 或 - 或 _ ，有空格也可以，不必连续。

三、换行

两个空格 + 一个回车。

四、代码

单行代码用1个`包裹
多行代码用3个`包裹，第一行可以说明语言类型，解析成标签
Tab或4个空格缩进形成的代码空间

五、引用

符号是 > ，引用里可以套引用，理论上无限嵌套。如：

一级

二级

三级

四级

六、强调

文字首尾各有一个*或者_为斜体
文字首尾各有两个*或者_为加粗
文字首尾各有三个*或者_为斜体加粗

七、链接

[页面文字](链接地址)，如：[百度](https://www.baidu.com/) 百度
![图片标题](链接地址)，如：[百度](https://www.baidu.com/img/bd_logo1.png)

但是md中不能新标签打开，所以常用：
<a href="www.baidu.com" target="_blank">百度xx</a>

八、列表

有序列表，数字加.和空格（注意，’ . ‘ 后面一定要跟一个空格）
无序列表，* - +加空格

支持嵌套，使用时在嵌套列表前按 tab 或 空格 来缩进,去控制列表的层数。

九、表格

用:的不同位置来改变对齐方式，默认左对齐(:-)，右对齐(-:)，居中对齐(:-:)

head head head
center left right
center left right
center left right

特殊使用

ToDo

  • 计划任务
  • 完成任务

另类超链接

点击[我]找到Baidu。
[我]: http://www.baidu.com

三种空格符：
&nbsp;：No-Break Space
&ensp;：en为字符单位，1Byte，等于em的一半宽
&emsp;：em为字符单位，2Byte，全角空格同一个汉字宽度
