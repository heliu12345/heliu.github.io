Java基础-语言基础（Basic)

一、Java基础说明

Java说明

一个Java文件可以有多个类但提倡只有一个
一个java文件只能有一个public class
Main函数是java的主入口PSVM进行主动运行
Main函数不属于成员函数，只是寄居在Class中
Main函数的参数args是启动java时附带的参数
JVM默认导入java.lang.*包

Java中的注释

//单行注释:从//开始本行结束
/* 开始 / 结束，多行注释
/* 开始 */结束，文档注释：javadoc会提取生成帮助文档，类方法常量

Java标示符

Java中的标示符组成：数字、字母、下划线、美元符
Java中的大小写敏感、没有长度限制，不可以用java关键字，不可用数字开头
