JDBC简介

介绍

  • Java Database Connectivity：Java访问数据库的解决方案
  • JDBC是SUN公司做的关于Java与数据库的接口
  • 具体实现是由各大数据库公司来制作，做出jar包称为连接数据库的驱动
  • 向对应数据库官网下载该jar包，导入到项目中

JDBC实现的接口

实现功能|对应接口
-|-|-
驱动管理|DriverManager
连接|Connection DatabaseMetaData
语句对象接口|Statement
PreparedStatement
CallableStatement
结果集接口|ResultSet
ResultSetMetaData

MySQL与Oracle实现

Oracle实现
  • ojdbc6.jar / ojdbc14.jar
  • Class.forName(“oracle.jdbc.OracleDriver”)

MySQL实现

  • mysql-connector-java-5.1.8-bin.jar
  • Maven可搜索jar包mysql-connector-java
  • Class.forName(“com.mysql.cj.jdbc.Driver”)

连接使用JDBC

传统使用步骤

  • 一般情况下我们在整个连接中都需要try,catch，其中1~5步在try中，第6步在finally中

  • 错误类型为SQLException，finally中close()或者使用java新语法try()定义三者　　　　

    //1.注册驱动
  Drive drive = new com.mysql.jdbc.Drive;    //最新com.mysql.cj.jdbc.Driver
  DriveManager.registerDriver(drive);

  //2.获取连接
  String url = "jdbc:mysql://localhost:3306/database";
  String user = "root";
  String password = "psd";
  Connection conn = DriverManager.getConnection(url,user,password);    
  //url连接最新版需要在地址后面增加`?serverTimezone=GMT%2B8`表示系统时区
  //定义写入字符集`?useUnicode=true&characterEncoding=UTF-8`

  //3.获取数据库操作对象
  Statement stmt = conn.createStatement();

  //4.执行SQL语句[DQL]
  String sql = "select * from table were a = 1";
  ResultSet rs = stmt.excuteQuery(sql);
  //执行SQL语句[DML]
  String sql = "insert into table values(1,2,3)";
  int count = stmt.executeUpdate(sql);
  //执行SQL语句
  boolean flag = stmt.excute(sql);
  //SQL语句结尾不用加分号，注意空格干扰语义

  //5.处理查询结果
  while(rs.next()){
  String a1 = rs.getString(1);        //以列的下标，JDBC下标都以1开始
  String a2 = rs.getString("col2");    //以列的名称
  }
  //next方法如果当前行有数据返回true，否则false。getXXX()取得字段内容

  //6.释放资源
  //先关闭ResultSet，后关闭Statement，最后关闭Connection
  //三个关闭分别try,catch，在try中都先判断是否为空
  try {
      if (rs!=null) rs.close();
      if (ps!=null) ps.close();
      if (conn!=null) conn.close();
 } catch (SQLException e) {                
   e.printStackTrace();
 }

常用使用方法

  • 第一步可以通过mysql写的静态代码块进行注册驱动：
  • Class.forName(“com.mysql.cj.jdbc.Driver”);
  • 第一步在MySQL的5.1.6及以后的jar包中自动进行注册驱动，可以不用写
  • 关于url的书写写法：
  • Oracle写法：jdbc:oracle:thin:@localhost:1521:sid
  • SQLServer写法：jdbc:microsoft:sqlserver://localhost:1433; DatabaseName=sid
  • MySQL写法：jdbc:mysql://localhost:3306/sid
  • Mysql的url地址的简写形式：jdbc:mysql:///sid
  • url定义时区：?serverTimezone=GMT%2B8
  • url定义字符集：?useUnicode=true&characterEncoding=UTF-8
  • 第一、二步中的参数可以读取properties配置文件获取
  • 第三、四步中为了防止SQL注入可以更改预编译SQL语句
