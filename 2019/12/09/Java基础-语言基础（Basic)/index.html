2019-12-09 Java

Java基础-语言基础（Basic)

一、Java基础说明

Java说明

一个Java文件可以有多个类但提倡只有一个
一个java文件只能有一个public class
Main函数是java的主入口PSVM进行主动运行
Main函数不属于成员函数，只是寄居在Class中
Main函数的参数args是启动java时附带的参数
JVM默认导入java.lang.*包

Java中的注释

//单行注释:从//开始本行结束
/* 开始 */ 结束，多行注释
/** 开始 */结束，文档注释：javadoc会提取生成帮助文档，类方法常量

Java标示符

Java中的标示符组成：数字、字母、下划线、美元符
Java中的大小写敏感、没有长度限制，不可以用java关键字，不可用数字开头
Java命名规范：

规范 方式 例子
类名或接口名 每个首字母大写 HelloWorld
方法名 首字母小写，其他首字母都大写 getName
常量名 全部字母大写，用_连接 BLUE_COLOR
变量名 首字母小写，后面的首字母大写，其他首字母小写 tomScrop
包名 全部字母小写 com.tedu.cn

二、Java基本语法

输出语句：System.out.println("")
输入语句：Scanner scan=new Scanner(System.in)

命名 语法 备注
变量声明 <类型><变量名> int age； String name；
int age=18； String name=”张三”；
常量声明 使用final public final static int WIDTH=0；
创建对象 <类名><对象名>=new<构造函数>； Thread t=new Thread()；
if分支 if(条件判断){}
else if(条件判断){}
else{以上if条件均为假时运行此语句}		 不使用大括号默认认为下一个；前面的语句属于if-true分支内，效率比switch低，switch直接进入不是挨个判断
switch分支 switch(表达式){default:默认语句；
break；
case 常量：语句；
break；}		 case说明程序开始，break说明结束（退出分支），没有则case穿透，default可以不在最后面但是不要忘记break，byte/char/short/int/String(1.7以后支持）不支持小数布尔类型等
for循环 for(int i=0；i<9；i++){} for(;;){}默认死循环，()内多条语句用，但是注意第一块如果声明只能一个类型的多声明
while循环 while(布尔表达式){}
do{}while(布尔表达式)		 continue跳过本次循环继续下次循环
foreach遍历 for(Object o:数组/集合){} JDK1.5开始支持的，仅是编译认可，编译过程中数组改回for语句，集合改回迭代器语句
break xx；跳出xx标记的循环
continue xx；跳出本次循环进行xx标记循环的下次循环
